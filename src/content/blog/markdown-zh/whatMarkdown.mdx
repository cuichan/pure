## Admonitions 警示块
警示框，又称提示框，是一种非常适合在不中断文档流的情况下包含附加内容的工具。Material for MkDocs 提供了多种类型的警示框，并允许在其中嵌套任意内容。
### 使用方法
警示框使用简单的语法：块以 !!! 开始，后跟一个作为类型限定符的关键字。块内容紧接在下一行，并缩进四个空格。
```Markdown
!!! note

    这是没有设置标题的 note 类型警示块。
```
